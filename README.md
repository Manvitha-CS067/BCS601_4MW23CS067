## Project Description
**Milestone Vault** is a cloud-native Personal Milestone Tracker designed to help users document and manage their academic and professional achievements. The application features a cinematic **Glassmorphism UI** and provides immediate user feedback through interactive celebrations (confetti) upon successful logging.

As a full CRUD (Create, Read, Update, Delete) application, it demonstrates seamless integration between a Python-based backend and a persistent database layer, optimized for cloud hosting.

## Key Features
*   **Real-time CRUD:** Create, view, edit, and delete milestones instantly.
*   **Time Tracking:** Integrated date-picker to log achievements by specific dates.
*   **Premium UX:** Interactive pop-ups via SweetAlert2 and visual celebrations via Canvas-Confetti.
*   **Responsive Design:** Fully mobile-responsive UI built with Tailwind CSS.

## Tech Stack
*   **Backend:** Python 3.x, Flask
*   **Database:** SQLAlchemy (SQLite)
*   **Frontend:** HTML5, Tailwind CSS, JavaScript
*   **Deployment:** Render (Cloud PaaS)
*   **WSGI Server:** Gunicorn
